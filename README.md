# Sleep Calculator for iOS
This is a simple sleep calculator. No user data that is not essential is collected. Clean, clear, light weight, and simple. 

I have intentionally chosen a small scope project as I want to build a tool I actually use and to leverage AI to operate as a guide, not as a do-er. 

Many repositories document software, but I want this repo to document my progress through the engineering process and design decisions behind the software. 

## Why
I have been looking for a tool to replace a deprecated sleep calculator app from the app store, but no alternatives fit my criteria. I want to refresh some skills, so this seemed like a perfect project to create. 

## Project Goals and Philosophy
It has been a while since I've developed in languages aside from Python and Mathematica, so this is a low stakes way to re-integrate myself into different tech stacks. 

The guiding philosophies are:
 - Clear Ownership: I want to understand and own every engineering decision in this repo. 
 - Sound and Defensible engineering judgement: I want to be sure that the trade-offs are well considered. 
 - Continuous Learning: The best way to learn is by doing, and by doing I will keep learning.
 - AI augmentation: I want to leverage AI to advance my learning, not to replace it. I want to be sure I understand 100% of this repo.
