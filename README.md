# Sleep Calculator for iOS
a SwiftUI sleep-cycle calculator developed to aid my understanding of SwiftUI and to relearn the basics of git/github

## Project Goals
This repository serves two primary purposes
1. Build a funcitonal and polished sleep calculator for iOS that is simple, clean and does not harvest user data. This is meant to be as basic as possible. 
2. Document the coding and engineering principles I learn and am reviewing along the way with a particular focus on leveraging AI to provide me with some good design questions. 

This project intends to emphasize understanding over blind imitation or AI regurgitation. AI is used to help structure course assignments and feedback, but most design is controlled by myself.
